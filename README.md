Our analyses required three modeling steps. 
First we calculated the best dilution parameter (d; Bijma, 2010) in 02-Dilution parameter for models with only helpers, moms and helpers, dads and helpers, and all three social partners
Second we tested all variance components using likelihood ratio tests and fixed effects using Wald F-statistic tests in 03-SignificanceTesting.
Finally, our primary results were calculated and modeled in 04-Modeling.
